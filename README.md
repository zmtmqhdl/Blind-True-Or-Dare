## 🖥️ BlindTrueOrDare
실시간으로 같은 방의 사람들과 익명으로 질문과 O/X 게임을 즐기는 실시간 게임 앱으로 웹소켓을 학습하기 위해 시작한 프로젝트입니다. 실시간 통신 구조를 이해하고자 WebSocket 기반 서버를 직접 구축했으며, REST API 및 WebSocket 서버를 내부/외부 환경으로 분리해 관리했습니다.  앱에서는 Build Variant를 활용해 엔드포인트를 구분함으로써 테스트 및 배포 환경 전환을 효율적으로 처리했습니다.
